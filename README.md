# 🚀 Code Card Generator

Create beautiful, shareable code snippet cards for social media in seconds.

**Code Card Generator** is a lightweight web application that helps developers turn code snippets into visually appealing social-media-ready cards. Customize your profile, add content, insert code, and export high-quality PNG images instantly.

🌐 **Live Demo:** https://cardgen.me/

---

## ✨ Features

- 📸 Upload a profile picture
- 👤 Customize display name and username
- ✅ Add a verified badge
- 📝 Include descriptions, tips, or captions
- 💻 Interactive code editor powered by CodeMirror
- 🎨 Syntax highlighting with Highlight.js
- 🌙 Dark and ☀️ Light export themes
- ⚡ Real-time preview updates
- 📥 Export cards as high-resolution PNG images
- 📱 Social-media-friendly design

---

## 🖼️ Preview Workflow

1. Upload your profile picture
2. Enter your display name and username
3. Add a programming tip or caption
4. Write or paste your code snippet
5. Choose a theme (Dark or Light)
6. Download the generated card as a PNG

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/code-card-generator.git

cd code-card-generator
```

### Run Locally

Simply open the project using a local server.

Example:

```bash
npx serve .
```

or

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```
